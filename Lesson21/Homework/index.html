There are few ways to define font-size:

  • px are good if you need strict definition for a size of the text. In other cases it is better to use relative units listed below.
  • em based on current font size
  • rem based on current font size
  • % based on html font-size
  • vh (viewport height) based on screen height
  • vw (viewport width)based on screen width
